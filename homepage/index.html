Bem vindo ao orcamento.dados.gov.br

orcamento.dados.gov.br contém os dados da Lei Orçamentária Anual - LOA de 2012 publicados como dados ligados.

O melhor ponto para iniciar a navegação dos dados é http://orcamento.dados.gov.br/id/item-de-despesa o qual retorna uma página listando os Itens de Despesa do orçamento federal de 2012.

Listar Ações

Uma lista dos padrões de URI aceitas por este serviço podem ser encontradas aqui. Alguns exemplos com descrição são fornecidos abaixo:

URI Descrição
http://orcamento.dados.gov.br/id/item-de-despesa lista os itens de despesa da LOA 2012
http://orcamento.dados.gov.br/doc/item-de-despesa.html?_page=0&_sort=-loa_valorDotacaoInicial lista todos os itens de despesa ordenados descendentemente pelo valor
http://orcamento.dados.gov.br/doc/item-de-despesa.html?loa_temLocalizador.label=No Estado de São Paulo lista todos os itens de despesa onde o localizador é "No Estado de São Paulo"
http://orcamento.dados.gov.br/doc/item-de-despesa.html?_pageSize=50&loa_temLocalizador.label=No Município do Rio de Janeiro - RJ&_sort=-loa_valorDotacaoInicial lista todos os itens de despesa onde o localizador é "No Município do Rio de Janeiro - RJ" ordenados descendentemente pelo valor
http://orcamento.dados.gov.br/doc/acao lista todas as Ações da LOA 2012
