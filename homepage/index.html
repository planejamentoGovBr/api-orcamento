education.data.gov.uk contains a snapshot of Edubase taken in 2009 and published as linked data.

The best place to start exploring the data is http://education.data.gov.uk/id/school which return a page listing schools within the dataset.

A list of the URI patterns supported by this site can be found here. A few samples along with descriptions are given below:

URI Description
http://education.data.gov.uk/doc/school lists schools within the dataset.
http://education.data.gov.uk/doc/school?_view=location lists schools within the dataset, showing their position on a map.
http://education.data.gov.uk/doc/schoo/phase/primary?districtAdministrative.label=Bristol, City of lists primary schools associated within the administrative district known as "Bristol, City of"
http://education.data.gov.uk/doc/school/phase/secondary?gender.label=Mixed lists mixed secondary schools
http://education.data.gov.uk/def/school/Gender_TERM/instance list the gender terms used in the dataset
http://education.data.gov.uk/def/school/School/property.html?_pageSize=50 lists the properties used in the dataset to describe schools. The list is paged with 50 properties per page.
http://education.data.gov.uk/def/school?_pageSize=50 lists the defined classes and properties used to describe schools in the dataset. The list is paged with 50 properties per page.
